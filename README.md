# Backend Setup
## Install Dependencies
`cd backend` and run `npm install` to install all dependencies.

## Create .env
Create a `.env` file in the backend root folder. Then declare PORT and MONGO_URI.

## Run
Run `npm start` to run it.

# Frontend Setup
## Install Dependencies

## Run
